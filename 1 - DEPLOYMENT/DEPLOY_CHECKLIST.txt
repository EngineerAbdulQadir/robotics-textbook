  DEPLOYMENT CHECKLIST - Your Own GitHub Pages
═══════════════════════════════════════════════════════════

□ STEP 1: Update Configuration (book/docusaurus.config.ts)
  
  Change these lines:
  ✎ url: 'https://YOUR_USERNAME.github.io'
  ✎ organizationName: 'YOUR_USERNAME'
  ✎ projectName: 'YOUR_REPO_NAME'
  
  Example:
  ✎ url: 'https://AQI.github.io'
  ✎ organizationName: 'AQI'
  ✎ projectName: 'robotics-textbook'

───────────────────────────────────────────────────────────

□ STEP 2: Create GitHub Repository
  
  1. Go to: https://github.com/new
  2. Name: robotics-textbook (or your choice)
  3. Make it PUBLIC
  4. Don't add README
  5. Click "Create repository"

───────────────────────────────────────────────────────────

□ STEP 3: Push Your Code
  
  Run in PowerShell:
  
  cd C:\Users\AQI\OneDrive\Desktop\HACKATHON\physical-ai-textbook-main
  git init
  git add .
  git commit -m "Initial commit"
  git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
  git push -u origin main

───────────────────────────────────────────────────────────

□ STEP 4: Enable GitHub Pages
  
  1. Go to repository Settings
  2. Click "Pages" in sidebar
  3. Source: "GitHub Actions"
  4. Done!

───────────────────────────────────────────────────────────

□ STEP 5: Wait for Deployment
  
  1. Go to "Actions" tab
  2. Watch deployment progress
  3. Wait for green checkmark ✅
  4. Takes 2-5 minutes

───────────────────────────────────────────────────────────

□ STEP 6: Access Your Site!
  
  URL: https://YOUR_USERNAME.github.io/robotics/
  
  Example: https://AQI.github.io/robotics/
